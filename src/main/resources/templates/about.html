About ContactDiary

ContactDiary is a secure and smart contact management web application built using Spring Boot, Thymeleaf, and MySQL. It helps you organize, search, and manage your contacts seamlessly in the cloud.

🔐 Features

  • Create and manage user accounts with email verification
  • Secure login with OAuth (Google, GitHub)
  • Smart contact storage and search
  • Export contacts to Excel
  • Mark favorites, edit profiles, and more

🌐 Vision

Our goal is to help individuals and teams manage contacts efficiently, securely, and with complete control — wherever they are. Whether you're a freelancer, a sales rep, or a student, ContactDiary is made for you.

💡 Technologies Used

Java · Spring Boot · Thymeleaf · MySQL · JDBC · Tailwind CSS · VS Code · Ubuntu
